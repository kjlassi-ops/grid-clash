# Grid Clash

Grid Clash is a casual Android game playable:
- Solo against AI
- Local multiplayer over Wi-Fi (same network)

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture

## Features
- Solo mode with AI (3 difficulty levels)
- Local multiplayer (host / join via Wi-Fi)
- Smooth mobile UI
- Fast gameplay
